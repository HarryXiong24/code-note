## 3. 回文子串

- 来源： leetcode 647

- 思路： 动态规划，
  > - dp[i][j]： 表示 str[i] -- str[j] 之间的字符串是否为回文串
  > - dp[i][i] = 1
  > - dp[i][j] = 1 if dp[i+1][j-1]== 1 and str[i] == str[j] else 0
